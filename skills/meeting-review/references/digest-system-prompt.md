# Meeting Digest System Prompt Reference

Reference material for the meeting digest extraction process.
See `tools/generate-meeting-digest/system-prompt.md` for the active LLM system prompt.

## Output Sections

- **Insights**: Durable findings with supporting evidence from the transcript
- **Action Items**: Tasks with owner, due date, status, and source quote
- **Decisions**: Resolved choices with rationale and source
- **Open Questions**: Unresolved items with owner and source
